# 摇摇晃摇

- 本项目基于 uni-app / Vue2 / JavaScript / Scss 开发的记账类型微信小程序

- 本小程序属于个人学习的开源项目。

- 小程序对接 [uniCloud](https://unicloud.dcloud.net.cn) 云服务器存储数据

- 使用 [uni-ai](https://uniapp.dcloud.net.cn/uniCloud/uni-ai-intro.html) 实现AI聊天
